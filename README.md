# Muhammad Faizan — Portfolio

This repository contains the source code for my personal portfolio site. It showcases who I am, the services I offer and the projects I’ve built as a Full‑Stack MERN developer. The site is built with **Next.js**, **TypeScript** and **Tailwind CSS** using the modern App Router. Feel free to fork or reuse the structure for your own portfolio.

## ✨ Features

- Responsive, accessible design with dark‑mode aesthetic.
- Hero, About, Skills, Projects, Services and Contact sections.
- Projects stored in a separate data file for easy updates.
- Tailwind CSS for styling with a custom primary colour.
- Ready to deploy on Vercel or any Node server.

## 📂 Project Structure

- `app/` – Contains the root layout and page components.
- `components/` – Reusable UI components such as Navbar, sections and Footer.
- `data/projects.ts` – Array of project objects used to populate the Projects section.
- `public/images/` – Project screenshots used in the portfolio.
- Configuration files: `next.config.js`, `tailwind.config.js`, `tsconfig.json` etc.

## 🚀 Getting Started

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Run the development server:**

   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser. You should see the portfolio running.

3. **Build for production:**

   ```bash
   npm run build
   npm start
   ```

## 📦 Deploying

The easiest way to deploy a Next.js app is via [Vercel](https://vercel.com). Simply sign in with your GitHub account, import this repository and follow the prompts. Vercel will automatically build and deploy your site with optimal settings.
